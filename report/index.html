The goal of this project to get the lexemes of the input Java code & identify each token & put it in the symbol table.


The Program works in this flow:

  • 1. Parse the lexical Input file

    It evaluates the regular defenitions & provide the right regular expression for the language.

    letter = a-z|A-Z

digit = 0-9
digits = digit+
num: digit+ | digit+.digits( \L|E digits )

id:(letter|_|$)(letter|digit|_|$)*

{abstract assert*** boolean break ...etc}
[; , \( \) { } \[ \] .]


equals = \=\=
notequals = !\=
larger = > | >\=
smaller = < | <\=

relop: equals | notequals | larger | smaller

assign: \=
addop: \+ | \-
mulop: \* | /
  • 2. There are 3 results of the first step: Keywords, Punctuations, Regular Expressions.

    Keywords

    while int if boolean break ...

    Punctuations

    ; , . { } [ ] ( )

    Regular Expressions

    assign: \=
num: (0-9)+ | (0-9)+.((0-9))+( \L|E ((0-9))+ )
  • 3. Generate the proper NFA for the keywords, punctuations & the regular Expressions
  • 4. Generate the DFA from the NFA provided by step 3
  • 5. Minimize the DFA
  • 6. Provide the final Lexical Result with errors (if exists) & putting them in symbol table.
